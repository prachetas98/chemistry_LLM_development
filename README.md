**ChemLLM**

**Brief Description:**
Web-based ChemLLM application for answering chemistry questions using transformer-based models, with support for training, inference, and API-based deployment.

**Technologies Used:**

* Python
* TensorFlow / PyTorch
* Transformers (BERT, RoBERTa, MISTRAL-7B, LLAMA, LSTM)
* Flask (web interface & REST API)
* NumPy, Pandas
* Docker
